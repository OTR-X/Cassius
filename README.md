### Install ⬇️

```bash
> apt update
> apt upgrade
> pkg update
> pkg upgrade 
> pkg install bash
> pkg install libwebp
> pkg install git -y
> pkg install nodejs -y 
> pkg install ffmpeg -y 
> pkg install wget
> pkg install imagemagick -y
> git clone https://github.com/vitya
> cd valera
> npm i
> npm start
```

#### Change `config.json` 
##### Register in <b>[zeks.xyz](https://zeks.xyz)</b> to get apiKey then enter to zeksKey in `config.json`
### Start ✅

```bash
> npm start [session name] [--options]
```
`options`
 `port`
--port [port number]
